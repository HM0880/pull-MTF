Motivation¶

Note

This code works on Windows 7, 32-bit and 64-bit.

Optikos LensCheck benches produce .thf (through-focus) MTF files. This GUI was designed to be an easy and flexible yet powerful way to plot any number of these .thf files in a given folder.

The two main files are process_THF_file.py and plot_MTF_GUI.py. Sphinx auto-generated documentation of the process_THF_file.py is <here <https://htmlpreview.github.io/?https://github.com/HM0880/pull-MTF/blob/master/_build/html/%2Bautodoc/process_THF_file.html>`__.

Four sample data files are in this repository’s data folder.
